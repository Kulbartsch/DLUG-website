Notizen - Apps und Konzepte

Eine alphabetisch sortierte Liste von "Note-taking" Apps und Konzepten.

Teilweise sind die Einträge mit Details versehen, meist die, die mich am meisten interessiert haben. Zu einigen Apps gibt es nur einen Link. Ergänzungen werden gerne entgegen genommen.

Am Ende des Artikels gibt es noch eine Liste von Artikeln und Blogs zum Thema Notizen.

Johnny.Decimal (Konzept)

A system to organise your life
  • https://johnnydecimal.com/
  • Anmerkung: Johnny.Decimal ist ein Konzept wie man Daten strukturiert ablegt, unabhängig von ihrem Speicherort (Dateien, Mail, Notizen, Notiz-Apps)
Johnny.Decimal is designed to help you find things quickly, with more confidence, and less stress.

Joplin

  • https://joplinapp.org/
  • Platform: Linux, Windows, macOS, Android, iOS, für Desktop Systeme sogar mit einer Terminal APP
  • Lizenz: MIT
  • Programmiersprache: Typescipt + Javascript
  • Plain Text: nope -> SQL (SQLite?)
  • Sync: über API oder Sync Verzeichnis
The notes can be synchronised with various cloud services including Nextcloud, Dropbox, OneDrive, WebDAV or the file system (for example with a network directory). When synchronising the notes, notebooks, tags and other metadata are saved to plain text files which can be easily inspected, backed up and moved around.
  • Anmerkung:
    • Die Dateien können verschlüsselt werden.
    • Joplin verwaltet die Dokumente selbst in .config in einer SQL Datenbank. Für den Sync werden die Dateien bei Bedarf exportiert. Die Dateien bekommen kryptische Namen.

The notes are stored as plain text markdown files and are synced with Nextcloud's/ownCloud's file sync functionality. Of course other software, like Syncthing or Dropbox can be used too. You can use your existing text or markdown files, no need for an import most of the times

  • https://zim-wiki.org/index.html
  • Platform: Linux, Windows, macOS
  • Lizenz: GPL
  • Programmiersprache: Python

    • ZIM unterstützt diverse Plugins, u.a. Graphviz
