Über Linux User Groups

Was ist eine LUG?

LUG steht für Linux User Group, also eine Gruppe von Linux Usern.

Über Linux User Groups aus Wikipedia, der freien Enzyklopädie

Eine Linux User Group, kurz LUG, ist eine Gruppe von Linux-interessierten Personen, die sich regelmäßig treffen. Die meisten LUGs finden sich in Städten oder Kreisen und veranstalten regelmäßig Treffen oder organisieren Veranstaltungen rund um Linux, wie beispielsweise Installationspartys. Jedoch sind die Themen meist nicht auf Linux beschränkt, sondern Linux steht als Symbol für jegliche freie Software.
